Keyboard Shortcuts

If you see a menu, menu item, or button with an underlined letter in its name, you can select that item by holding down the Alt key and simultaneously pressing the letter that is underlined. For example, open the Actions menu by pressing Alt+A.

You can also use the following additional keyboard shortcuts:
